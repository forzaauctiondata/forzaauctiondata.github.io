About the Auction Data

Data is regularly collected from an Auction House search for all cars

The information on this site is routinely collected from an unfiltered Auction House search for all cars. Each entry represents a car that was successfully auctioned. 

The auctions that are returned by that search complete within about two minutes. Each auction is saved once it completes. The information here should represent random sample of all the cars that are sold there.

This dataset doesn't include "buy out" auctions that end more than about two minutes earlier than the scheduled auction end-time, because the Auction House search won't include auctions that have already finished. It does include auctions that were bought out right before the scheduled auction end-time.

About Interacting with the Plots

This site uses interactive Bokeh plots to present data

These plots are responsive and interactive. View data within a specific date range and price by using the plot tools (located in the top right corner of the page) to pan, zoom, and scale data on the plots. Use the interactive legend to show or hide data for specific car models.
